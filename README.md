# 🎞 Rimflix

## Rimflix 소개

> 넷플릭스를 모티브로 만든 영화 사이트입니다. 현재 상영하고 있는 영화나 인기있는 컨텐츠들을 볼 수 있으며, 또한 영화 검색 기능를 검색할 수 있습니다. 또한 배너에 있는 영화는 트레일러 영상도 볼 수 있습니다.
>
> 이렇게 여러 영화와 텔레비젼 컨텐츠들을 둘러볼 수 있는 것에 더해, 내가 좋아하는 영화나 TV를 저장하여 따로 볼 수 있는 공간도 개발하였습니다.

## 프로젝트 후기
이번 프로젝트에서 react-query 사용해봤는데, 정말 장점을 많이 느꼈습니다. 일단 몇줄의 코드로 쉽게 데이터 fetch가 가능한 것부터가 장점이었는데 로딩 부분도 자체적으로 제공해주기 때문에 코드의 양이 줄고, 깔끔해져서 좋았습니다. 가장 큰 장점은 아무래도 캐싱 부분이었습니다. react-query의 캐싱 능력 덕분에 앱의 속도가 더 원활한 것을 체감했습니다. 앞으로도 정말 잘 사용할 것 같습니다.🥰

넷플릭스를 바탕으로 림플릭스를 만들어봤는데 정말 재미있었던 작업이었습니다. 이 프로젝트를 통해서 framer-motion 애니메이션 라이브러리 사용해도 조금 더 익숙해졌습니다. 전체적으로 애니메이션을 사용하니 정말 사이트가 생동감이 있다는 느낌이 들었습니다. 앞으로 애니메이션 라이브러리도 자주 사용할 것 같습니다.🤓

지금 프로젝트를 완료했다고 하지만 몇가지 재미있는 기능들을 더 추가해볼 예정입니다.😌
