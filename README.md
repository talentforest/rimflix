# 🎞 Rimflix

## Rimflix 소개

> 넷플릭스를 모티브로 만든 영화 사이트입니다. 현재 상영하고 있는 영화나 인기있는 컨텐츠들을 볼 수 있으며, 또한 영화 검색 기능를 검색할 수 있습니다.
>
> 이렇게 여러 영화와 텔레비젼 컨텐츠들을 둘러볼 수 있는 것에 더해, 내가 좋아하는 영화나 TV를 저장하여 따로 볼 수 있는 공간도 개발하였습니다.
